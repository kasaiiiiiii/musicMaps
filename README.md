# 🎵 オンラインカスタム譜面ストレージ (Maps)

このリポジトリは、自作音ゲーの**カスタム難易度（Customモード）**で読み込まれる譜面データを管理・配信するためのストレージです。
ノーツエディターで出力した譜面ファイル（JSON）と、その目次となる `index.json` を曲ごとのフォルダに分けて配置することで、ゲーム本体をアップデートすることなくリアルタイムに新規譜面をプレイヤーへ配信できます。

---

## 📁 フォルダ構成のルール

`Maps/` フォルダの直下に、**ゲーム内の曲名（TrackName）と完全に一致するフォルダ**を作成し、その中にデータを配置してください。

```text
Maps/
 ├── (曲名フォルダ1)/
 │    ├── index.json          # フォルダ内の譜面リスト（必須）
 │    ├── Map_Easy.json       # 実際の譜面データ1
 │    └── Map_Hard.json       # 実際の譜面データ2
 └── (曲名フォルダ2)/
      ├── index.json
      └── Map_Crazy.json
```

## 📝 index.json の詳細な記述形式とルール

ゲームの選曲画面（スクロールリスト）にカスタム譜面の名前を正しく表示させるため、`index.json` は以下のルールに従って記述してください。

### 📋 基本テンプレート（コピペ用）
1つのフォルダに複数の譜面がある場合は、以下のように半角カンマ `,` で区切って並べます。

```json
{
  "maps": [
    {
      "displayName": "譜面名1（ゲーム内で表示される名前）",
      "fileName": "ファイル名1.json"
    },
    {
      "displayName": "譜面名2",
      "fileName": "ファイル名2.json"
    }
  ]
}
```
